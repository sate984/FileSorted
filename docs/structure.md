# Вклад в проект

Проект **FileSorted** разрабатывается в рамках учебной группы **ИСП11-Kh21**.  
Основной (и пока единственный) разработчик: **[sate](https://github.com/sate984)**.

## Текущий статус
На данный момент проект **не принимает внешние pull request'ы** и разрабатывается индивидуально.

## Если вы участник учебной группы
- Все изменения согласовываются лично с разработчиком.
- При совместной работе:
  1. Создайте новую ветку: `feature/название-фичи`
  2. Реализуйте изменения
  3. Отправьте код на ревью

## Требования к коду
- Используется **Java 17+**
- Проект собирается через **Maven**
- Соблюдается структура пакетов: `com.example.sorter`
- Все новые функции должны сопровождаться обновлением документации в `/docs/`

## Сообщить об ошибке
Если вы нашли баг:
1. Откройте **Issue** в репозитории
2. Укажите:
   - ОС
   - Версию Java
   - Шаги для воспроизведения
   - Ожидаемый и фактический результат

# Руководство пользователя — FileSorted

## Как использовать приложение

1. Запустите приложение (`java -jar smart-file-sorter.jar` или через IDE).
2. Нажмите кнопку **«Выбрать папку»**.
3. В открывшемся проводнике выберите директорию, которую хотите отсортировать.
4. Приложение автоматически:
   - Проанализирует все файлы в папке.
   - Создаст подпапки: `images`, `docs`, `videos`, `audio`, `archives`, `others`.
   - Переместит файлы в соответствующие папки по расширениям.
5. Результат отобразится в логе внутри окна приложения.

## Поддерживаемые расширения

- **Изображения**: `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.svg`
- **Документы**: `.pdf`, `.doc`, `.docx`, `.txt`, `.rtf`, `.odt`
- **Видео**: `.mp4`, `.avi`, `.mkv`, `.mov`, `.wmv`
- **Аудио**: `.mp3`, `.wav`, `.ogg`, `.flac`
- **Архивы**: `.zip`, `.rar`, `.7z`, `.tar`, `.gz`
- **Остальное** — перемещается в папку `others`

> Если расширение не распознано, файл попадает в `others`.

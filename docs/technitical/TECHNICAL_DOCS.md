# Техническая документация — FileSorted

## Общая архитектура

Приложение состоит из двух основных классов:

- **`Launcher.java`** — точка входа (`main`-метод), необходимая из-за модульной системы JavaFX.
- **`FileSorterApp.java`** — основной класс, наследующий `Application`, содержит:
  - Логику UI (JavaFX: кнопки, лог-панель, выбор папки).
  - Бизнес-логику сортировки файлов с использованием **Java NIO** (`Files`, `Path`, `Files.move`).

## Модульность

- Проект использует **Java Platform Module System (JPMS)**.
- `module-info.java` подключает модули JavaFX (`javafx.controls`, `javafx.fxml`).
- Все зависимости управляются через **Maven** (`pom.xml`).

## Алгоритм сортировки

1. Получение пути к целевой директории через `DirectoryChooser`.
2. Рекурсивный обход файлов (только верхний уровень — без подпапок).
3. Для каждого файла определяется расширение.
4. Файл перемещается в соответствующую подпапку с помощью `Files.move()`.
5. Операции логируются в UI-панель в реальном времени.

> Обработка исключений: при ошибках (файл занят, нет прав) выводится сообщение в лог.
